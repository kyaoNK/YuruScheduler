# YuruScheduler

動画制作の工程管理を効率化するデスクトップアプリケーション

## 概要

YuruSchedulerは、動画制作者のための工程管理ツールです。公開日から逆算して各工程の締切を自動生成し、視覚的な警告機能でタスク管理をサポートします。

### 主な機能

- 📅 **公開日からの自動逆算**: 公開日を設定すると、工程ごとの締切が自動生成されます
- 🎨 **カスタマイズ可能な工程設定**: 初稿、最終稿など、自由に工程を追加・編集できます
- 📊 **2つの表示モード**: タイムライン表示とカレンダー表示を切り替え可能
- 🚨 **視覚的警告**: 締切が近づくと色とアイコンで警告
- 💾 **ローカルデータ保存**: データはすべてローカルに保存され、プライバシーを保護
- 🔄 **自動・手動カードの管理**: 自動生成されたカードを編集すると手動カードに変換

## 使い方

### 1. 最初の設定

アプリを起動したら、まず右上の「⚙️」ボタンから工程設定を行います。

デフォルトでは以下の工程が設定されています：
- **初稿締切**: 公開日の3日前
- **最終稿締切**: 公開日の1日前

### 2. カードの作成

1. 右上の「+ カード作成」ボタンをクリック
2. タイトル、公開日、締切日、メモ（任意）を入力
3. 「作成」をクリック

手動カードを作成すると、設定した工程に基づいて自動カードが生成されます。

### 3. 表示モードの切り替え

- **タイムライン表示**: 日付順にカードを縦1列に表示
- **カレンダー表示**: 月単位のカレンダーで予定を一覧

右上の「📊」ボタンで切り替えができます。

### 4. 視覚的警告

締切が近づくと、カードの色とアイコンが変化します：

| 状態 | アイコン | 色 | 条件 |
|------|---------|-----|------|
| 期限超過 | 🚨 | 濃い赤 | 締切を過ぎている |
| 緊急 | ⚠️ | 赤 | 0〜3日以内 |
| 警告 | ⏰ | 黄色 | 4〜7日以内 |
| 通常 | - | 白 | 7日以上 |
| 完了 | ✅ | グレー | 完了済み |

### 5. カードの編集と削除

- **編集**: カードの「編集」ボタンをクリック
- **削除**: カードの「削除」ボタンをクリック
- **完了**: チェックボックスをクリック

### 6. 自動カードの手動変換

自動生成されたカードを編集すると、手動カードに変換されます。手動カードは公開日を変更しても再計算されません。

**注意**: このアプリはローカルで動作し、データはすべてお使いのコンピュータに保存されます。インターネット接続は不要です。
